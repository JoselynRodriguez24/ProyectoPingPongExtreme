PROYECTO: Ping Pong Extreme

INSTRUCCIONES PARA EJECUTAR EL PROYECTO

1. Tener instalado Java JDK 17 o una versión superior.

2. Abrir el proyecto "ProyectoPingPongExtreme" en NetBeans.

3. Esperar a que el IDE cargue todas las librerías del proyecto.

4. Verificar que la clase principal sea en el paquete pingpong clase main.java

5. Ejecutar el proyecto presionando el botón o la tecla F6

6. En la pantalla de inicio, ingresar el nombre de los jugadores, seleccionar la dificultad y comenzar la partida.

CONTROLES

Jugador 1
- tecla W: Mover la paleta hacia arriba.
- tecla S: Mover la paleta hacia abajo.

Jugador 2
- Flecha Arriba: Mover la paleta hacia arriba.
- Flecha Abajo: Mover la paleta hacia abajo.

REQUISITOS

- Java JDK 17 o superior.
- NetBeans IDE.
- Sistema operativo Windows, Linux o macOS.

DESCRIPCIÓN

Ping Pong Extreme es un juego desarrollado en Java utilizando Java Swing. El objetivo es que dos jugadores compitan controlando una paleta para golpear las bolas, obtener puntos y ganar las rondas hasta convertirse en el ganador de la partida.
